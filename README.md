# Developer Portfolio

A minimalist portfolio website showcasing your projects and professional presence.

## Features

- 🌓 Dark/Light mode
- 📱 Fully responsive design
- 🚀 Dynamic GitHub projects integration
- ⭐ Project stars and publish dates
- 🔗 Social media links
- 🎨 Clean and modern UI

## Setup Instructions

1. Download all files from this package
2. Create a new repository named `yashrajnayak.github.io`
3. Upload the following files to your repository:
   - `index.html`
   - `styles.css`
   - `scripts.js`
4. Go to repository Settings > Pages
5. Under "Source", select "main" branch
6. Your portfolio will be live at: `https://yashrajnayak.github.io`

Note: It may take a few minutes for your site to be published.

## Customization

Feel free to modify the colors and styles in `styles.css` to match your preferences. The website uses CSS custom properties (variables) for easy theming.
