# Developer Portfolio

A minimalist portfolio website showcasing your projects and professional presence.

## Features

- 🌓 Dark/Light mode
- 📱 Fully responsive design
- 🚀 Dynamic GitHub projects integration
- ⭐ Project stars and publish dates
- 🔗 Social media links
- 🎨 Clean and modern UI

## Customization

Feel free to modify the colors and styles in `styles.css` to match your preferences. The website uses CSS custom properties (variables) for easy theming.
